# real-world-forecasting-foundations
This repo powers the Forecast Academy course that teaches forecasting the way it's actually done at scale-practical, decision-driven &amp; built w/Nixtla's modern time-series stack. You'll learn how to frame business problems, design evaluation plans, build baselines/ML models, &amp; create stable, trustworthy forecasts that real organizations rely on.
